# homebrew-tap

Homebrew tap for custom formulae by [@idvoretskyi](https://github.com/idvoretskyi).

## Usage

Add this tap:

```sh
brew tap idvoretskyi/tap
```

Then install any formula:

```sh
brew install idvoretskyi/tap/<formula>
```

## Formulae

| Formula | Description | Version |
|---------|-------------|---------|
| [`cncf-maintainers`](Formula/cncf-maintainers.rb) | CLI to validate GitHub usernames against the CNCF maintainers list | `0.1.0` |
